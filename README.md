# ราชาหวาย 👑

เกมไพ่ปาร์ตี้ไทย — บลัฟเพื่อครองบัลลังก์และลงโทษกบฎ

## วิธีเล่น

1. เพิ่มผู้เล่น 3–10 คน
2. แจกไพ่คนละ 1 ใบ แปะบนหน้าผากหงายหน้า (อย่ามองไพ่ตัวเอง)
3. รอบบลัฟ — ผลัดกันตัดสินใจเก็บไพ่หรือจั่วใหม่ (จั่วได้ครั้งเดียว)
4. นับถอยหลัง 3-2-1 แล้วเปิดไพ่
5. ไพ่สูงสุดเป็น **ราชาหวาย** ไพ่ต่ำสุดเป็น **กบฎ**
6. ราชาเลือกโทษให้กบฎ

### การจัดอันดับไพ่

- แต้ม: A > K > Q > J > 10 > 9 > ... > 2
- ดอก (ถ้าแต้มเท่ากัน): โพดำ > ข้าวหลามตัด > ดอกจิก > โพแดง

## เริ่มพัฒนา

```bash
npm install
npm run dev
```

เปิด [http://localhost:5173](http://localhost:5173) ในเบราว์เซอร์

## Build

```bash
npm run build
npm run preview
```
